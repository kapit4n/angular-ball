# Install Server
* clone project
* cd projectPath/
* npm install
* node .
* go to http://localhost:3000
* go to http://localhost:3000/explore

# Install Client
* clone project
* cd projectPath/client
* npm install
* ng serve
* go to http://localhost:4200
